# MazeJS

An interactive browser-based maze generation and visualization project built with vanilla HTML, CSS, and JavaScript.

## Overview

MazeJS explores algorithmic generation, graph traversal, and visualization in a lightweight, dependency-free web environment. The project emphasizes clean separation of concerns, deterministic grid modeling, and real-time rendering in the browser.

## Technical Highlights

- Implements algorithmic maze generation within a structured grid representation.
- Encodes the maze as an implicit graph, enabling traversal and pathfinding extensions.
- Modular architecture separating UI (index.html, main.js) from core generation logic (maze.js).
- Designed as a sandbox for experimenting with search strategies, algorithmic efficiency, and visualization of state-space exploration.

## Execution

Open index.html in a modern browser.
